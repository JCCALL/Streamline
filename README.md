# STREAMLINE

## Description of the Site

"Streamline" is a full stack application to help users find out where movies and tv shows are streaming.

This application was driven by a desire to spend less time trying to find out where you can watch a movie or tv show.  The app allows the user to save a watchlist and can give information on where the show is available to watch with the current streaming and purchasing platforms available.
